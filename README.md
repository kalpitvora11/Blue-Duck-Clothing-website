# 🦆 Blue Duck Clothing Website

## 📌 Overview

The **Blue Duck Clothing Website** is a web-based project designed to showcase a modern and responsive clothing store interface. It demonstrates core frontend development skills including layout design, styling, and user interaction.

---

## 🚀 Features

* 🛍️ Attractive and responsive UI
* 📱 Mobile-friendly design
* 🧾 Product display sections
* 🔍 Easy navigation between pages
* 🎨 Clean and modern styling

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout
* **JavaScript** – Interactivity (if used)

   git clone https://github.com/kalpitvora11/Blue-Duck-Clothing-website.git

   OR open it in any browser
